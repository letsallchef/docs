## Search Users

`GET /users`

**Parameters**

| Name | Type | Description |
| --- | --- | --- |
| query | String | Part of `username` or `first_name` of the user. |

{% highlight json %}

    "id": ID,

    "last_name": String,
    "username": String,
    "rating": Number
}
{% endhighlight %}
